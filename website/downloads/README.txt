AGIS Defence System
===================

Thank you for choosing AGIS Defence System! This README provides all the information you need to get started.

Installation Instructions
-----------------------
1. Extract the ZIP file to your desired location
2. Right-click on install.bat and select "Run as administrator"
3. Follow the installation prompts
4. After installation completes, you can start the system using either:
   - The desktop shortcut (created automatically)
   - The start.bat file in your installation directory

System Requirements
------------------
- Windows 10 64-bit or later
- Python 3.8 or later
- 4GB RAM minimum (8GB recommended)
- 2GB free disk space
- Internet connection for updates

What's Included
--------------
This installer package contains:
1. Core Application Files (agis_defence directory)
2. Installation Scripts:
   - install.bat - Automated installation script
   - start.bat - Application launcher
3. Dependencies File (requirements.txt)
4. Documentation

Security Features
----------------
- ZIP format distribution for transparency and security
- Isolated Python virtual environment to prevent conflicts
- Proper permission handling and validation
- Secure dependency management
- Automated environment setup

Installation Process
-------------------
The installer will automatically:
1. Verify system requirements
2. Create an isolated Python virtual environment
3. Install all required dependencies
4. Configure system settings
5. Create desktop shortcuts
6. Set up proper file permissions

Troubleshooting
--------------
If you encounter any issues:
1. Ensure you're running install.bat as administrator
2. Check that you meet all system requirements
3. Verify your internet connection
4. Make sure no antivirus is blocking the installation
5. Check the installation logs in the same directory
